Exercice 4 : Exercice : Recréer la méthode join() Objectif de l'exercice : En utilisant les connaissances sur les fonctions, vous devez recréer la méthode join() en Python. Instructions : Définissez une fonction appelée join(separateur, *elements). La fonction join doit accepter un séparateur en tant que premier argument, suivi d'un nombre variable d'éléments à joindre. À l'intérieur de la fonction, initialisez une variable resultat avec la valeur convertie en chaîne de caractères du premier élément. Utilisez une boucle pour parcourir les éléments à partir du deuxième élément. À chaque itération de la boucle, ajoutez le séparateur suivi de l'élément converti en chaîne de caractères à la variable resultat. À la fin de la boucle, retournez le résultat final obtenu en joignant tous les éléments avec le séparateur. Testez votre fonction en l’appelant avec différents séparateurs et éléments, puis affichez les résultats obtenus. Exemple d’utilisation : resultat = join(", ", "rouge", "vert", "bleu") print("Affiche :", resultat) # Affiche : rouge, vert, bleu resultat = join(" - ", 1, 2, 3, 4, 5) print("Affiche :", resultat) # Affiche : 1 - 2 - 3 - 4 - 5 resultat = join(" | ", "pomme", "banane", "orange") print("Affiche :", resultat) # Affiche : pomme | banane | orange Contraintes : Vous n'êtes pas autorisé à utiliser la méthode join() intégrée à Python pour cet exercice. Assurez-vous que votre fonction fonctionne correctement avec un nombre variable d'éléments. Consignes : Vous devez fournir le code complet de la fonction join(). Assurez-vous d'inclure des commentaires pour expliquer le rôle de chaque instruction dans votre code. Testez votre fonction avec différents cas de test pour vous assurer de sa précision et de sa fonctionnalité. Vous pouvez utiliser la cellule de code fournie pour écrire votre solution. Exercice 7 : Créer des classes de base avec les classmethod Notions abordées : les classes, les méthodes. On continue encore et encore avec notre classe Voiture, pour laquelle nous allons maintenant créer des 'classmethod'. Cela nous permettra donc de définir une marque, un prix et une couleur par défaut pour 2 types de voitures de luxe : une Lamborghini et une Porsche. Nous allons donc à partir de ces deux classmethod créer deux instances de voitures : 1. >>> Lamborghini = Voiture.lamborghini() 2. >>> print(lamborghini.prix) 3. 150000 4. >>> porsche = Voiture.porsche() 5. >>> print(porsche.prix) 6. 200000 La Lamborghini devra avoir comme marque "Lamborghini", comme prix 150,000€ et comme couleur "rouge". La porsche quant à elle devra avoir comme marque "Porsche", comme prix 200,000€ et comme couleur "noire". Le code devra afficher : La Lamborghini a pour marque : Lamborghini, prix : 150.000€ et couleur : rouge La Porsche a pour marque : Porsche, prix : 200.000€ et couleur : noire Astuces : Pour créer une classmethod, il faut utiliser le décorateur @classmethod.
